pivot

syntax: (pivot str-table str-rows str-columns str-values [str-aggregate functions bool-pretty-print])
parameter: str-table - table name or subquery which must be between parentheses
parameter: str-rows - 1 or more comma & space seperated string of row fields
parameter: year from field) and aliases with ' as '
parameter: (str-values> - 1 or more comma and space seperated string of (probably numeric) fields, you can also put in calculations, functions and aliases with ' as '
parameter: optional string of comma and space seperated aggregation function: sum, max, min, count, avg, defaults to sum
parameter: bool-pretty-print - optional, if true will print a pivottable to the terminal, if nil will return a list of lists with results, default is true

return: a pivoted list or pretty-printed pivot-table with header
